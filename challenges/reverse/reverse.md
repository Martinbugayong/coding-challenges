## reverse 

**Difficulty**: basic 

## Prompt 

Write a function called `reverse` that accepts an array **or** string as an argument and returns the reverse of that string or array.

## Examples

```js 
reverse([1,2,3,4,5]); //=> [5,4,3,2,1]
reverse([99, 3, 45]); //=> [45, 3, 99]
reverse("cat"); //=> "tac"
reverse("topiary bon bon"); //=> "nob nob yraipot"
```
