## or

**Difficulty**: Basic 

## Prompt 

Write a function called `or` that accepts two boolean (`true` or `false`) parameters. The `or` function should return `true` if one of the arguments is `true` and `false` otherwise.

## Examples

```js
or(false, true) //=> true 
or(true, false) //=> true 
or(true, true) //=> true 
or(false, false) //=> false
```
